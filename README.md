# MenuProxy

Made to conveniently toggle system HTTP/HTTPS/SOCKS proxies from the macOS menu bar. Handy when you need to quickly enable/disable proxies while developing or debugging network traffic, for example when using tools like `mitmproxy`.

![MenuProxy screenshot](docs/screenshot.png)

Quick start
- Open the Xcode workspace: `open MenuProxy.xcworkspace` (or generate it with `tuist generate`).
- Build & run the `MenuProxy` app in Xcode(or run with `tuist run MenuProxy`).
- Tests are available under the `MenuProxyTests` target and can be run from Xcode or via Tuist (`tuist test`).
